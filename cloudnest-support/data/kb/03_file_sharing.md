# File Sharing and Permissions

## Sharing a File or Folder

To share a file or folder, right-click it and select "Share". You will see three options:

1. **Share link** — generates a URL anyone with the link can use. You can set the link to view-only or allow editing.
2. **Share with people** — invite specific email addresses. Recipients must have a CloudNest account (free is fine) and will see the file in their "Shared with me" section.
3. **Embed** — generates an iframe snippet you can paste into a website or blog. (Pro and Business plans only.)

## Permission Levels

- **Viewer** — can see and download the file
- **Commenter** — can see, download, and add comments (documents and images only)
- **Editor** — can modify the file and re-upload changes
- **Owner** — full control including deleting and changing other people's permissions

## Link Expiry and Passwords

On Personal plans and above, you can set:
- An expiry date for share links (1 day, 7 days, 30 days, or custom)
- A password required to access the link
- A download limit (e.g. revoke after N downloads)

Free plan share links never expire and cannot be password-protected.

## Revoking Access

To revoke access, right-click the file → Share → Manage Access. You can remove individual people or disable the entire share link. Once revoked, anyone with the link will see a "This link is no longer available" message.

## Sharing Limits

- Free: 100 active share links per account
- Personal: 1,000 active share links
- Pro: 10,000 active share links
- Business: unlimited

If you hit the limit, you'll need to revoke older links before creating new ones.
